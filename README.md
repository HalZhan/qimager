# QImager
- One minimal tool for image processing.
- 极简图像处理工具

# Steps
# 步骤
- npm install webpack -g
- 全局安装webpack
- npm install
- 安装项目依赖
- npm start
- 启动项目
- visit localhost:3377 with Chrome
- 打开Chrome并访问localhost:3377

# 文档

- 基于Angular 2

- 使用typescript进行开发
