# QImager
- One minimal tool for image processing.
- 极简图像处理工具

# Steps
# 步骤
- npm install webpack -g
- 全局安装 webpack
- npm install webpack-dev-server -g
- 全局安装 webpack-dev-server
- npm install
- 安装项目依赖
- npm start
- 启动项目
- visit localhost:80 with Chrome
- 打开Chrome并访问localhost:80
